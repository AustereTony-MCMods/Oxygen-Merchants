0.10.0b [2020-01-17]
+ Доработан дизайн меню.
+ Убрана возможность привязки профиля торговца к сущности, открыть меню для игрока теперь можно с помощью команды.
+ Добавлена поддержка выбора используемой виртуальной валюты для профиля торговца.
+ Добавлен таймаут для операций в меню для предотвращения исполнения неавторизованных запросов (запросы синхронизации, покупка/продажа). 
Таймаут индивидуален для каждого игрока и сбрасывается при открытии меню с помощью команды.
+ Добавлен кулдаун в 1 секунду для отправки пакетов - операций в меню управления (редактирование профилей).
+ Добавлен кулдаун в 500 миллисекунд для отправки пакетов - операций в меню торговца (покупка/продажа).
+ Обновлены параметры конфигурации:
 Сервер 
 - "merchant_menu_operations_timeout_millis" - [int] - таймаут операций в меню торговца в миллисекундах
 - "allow_ingame_profiles_management" [boolean] - разрешение на редактирование профилей торговцев во время игры
 
+ Добавлены команды:
 Сервер - Операторы [/oxygenop]
 - Открыть меню управления: /oxygenop merchants -management
 - Открыть меню торговца: /oxygenop merchants -merchant-menu <target> <profile id>
 
 Клиент [/oxygenc]
 - Сбросить данные клиента: /oxygenc merchants -reset-data
